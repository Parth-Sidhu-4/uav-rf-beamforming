# Security Policy

## Supported Versions

This is an academic research project and is **not actively maintained**. 
There are no supported versions, and no security updates will be provided.

| Version | Supported          |
| ------- | ------------------ |
| All     | :x: Not supported  |

## Reporting a Vulnerability

Because this is a closed, proprietary, simulation-only academic repository, we do not have a bug bounty program or a vulnerability remediation process. 

The code provided here is intended for offline simulation and study in controlled academic environments. It is **expressly prohibited from being deployed in operational UAV or RF systems** (see `DISCLAIMER.md`).

If you still feel the need to report a critical issue, you may email the author at parthecstasy@gmail.com, though a response or patch is not guaranteed.
